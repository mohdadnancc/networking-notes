## 📖 What I Learned

- Every device on a network has a unique IP address.
- Public IP is used on the Internet, while Private IP is used inside local networks.
- IPv4 uses 32-bit addresses.
- DNS converts domain names into IP addresses.
- DNS primarily uses Port 53.
- ping is used to test network connectivity.
- nslookup is used to find the IP address of a domain.
- HTTP is used for web communication.
- HTTPS is the secure version of HTTP.
- SSH is used for secure remote login.
